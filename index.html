Milieumetingen in je buurt

Waterkwaliteit in Vlaanderen en luchtkwaliteit in heel België. Klik een meetpunt op de kaart, of zoek op gemeente, nummer of waterloop.

Databronnen

    Meetplaatsen: Digitaal Vlaanderen. Waterresultaten: Vlaamse Milieumaatschappij. Luchtmetingen: IRCELINE. Deze toepassing is geen officiële website van een van hen.

    De broncode staat op GitHub. Klopt er een cijfer of een uitleg niet, of mis je een databron of een functie? Meld het als issue; daar heb je geen code voor nodig. Werk je zelf een verbetering uit, dan is een pull request welkom.
